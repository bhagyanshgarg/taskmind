🧠 TaskMind

Automatic timesheet generator & activity tracker for Linux.
Privacy-first. No cloud. No GPU. Under 100MB RAM.

⭐ GitHub Repo 📖 Documentation

📊 Auto Timesheet

Tracks your window activity and generates timesheets automatically. Never forget what you worked on.

🎙️ Meeting Recording

Auto-detects Meet/Zoom/Teams and records both sides. Compressed to tiny OGG files (~1.7MB/hr).

🔒 100% Private

Runs entirely on your machine. Zero cloud, zero network calls, zero telemetry. SQLite database.

🪶 Lightweight

Under 100MB RAM. No GPU. Works on 4GB machines. Ubuntu 18.04+ compatible.

📈 Web Dashboard

Beautiful dark-theme dashboard at localhost:7890. Today, weekly, monthly views with charts.

🔌 Export Anywhere

Export to CSV, JSON, Clockify, Toggl, or Jira Tempo. Fits your existing workflow.

Quick Install

git clone https://github.com/rishapgandhi/taskmind && cd taskmind && bash install.sh

One command. Interactive setup. Ready in 30 seconds.
